Introduction to TypeScript

TS crash course by Arin Bagul

TypeScript is a free, open source, high level programming language, developed by Microsoft. It is developed to deal with the shortcoming of JavaScript. The extension of a TypeScript file is .ts. It is the superset of JavaScript, means every correct JavaScript code can be compiled in TypeScript but can produce some warnings. It's main feature is that, every entity has a type.

We will dive into the topic of basic types in next chapter: Basic Types
